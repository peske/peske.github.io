DetailedExceptionsIndicates if the exceptions will be logged with all details or simplified. If false the simplified method will be used which basically logs output of System.Exception.ToString method. If true the System.Exception instance will be logged with all its properties. Defaults to false. Setting this property to true can cause significantly larger log entries. It also includes additional workload in serializing the exceptions which can affect performances in system with many exceptions. According to roughly measurement serialization of one System.Exception instance takes about 1ms on a decent machine. (Inherited from FD.Diagnostics.NLog.ElasticSearch.ElasticsearchTarget)
