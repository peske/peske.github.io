Public Properties
 NameDescription
DetailedExceptionsIndicates if the exceptions will be logged with all details or simplified. If false the simplified method will be used which basically logs output of System.Exception.ToString method. If true the System.Exception instance will be logged with all its properties. Defaults to false. Setting this property to true can cause significantly larger log entries. It also includes additional workload in serializing the exceptions which can affect performances in system with many exceptions. According to roughly measurement serialization of one System.Exception instance takes about 1ms on a decent machine.  
ElasticsearchSerializer Gets or sets an alternative serializer for the ElasticSearch client to use.  
Fields Gets or sets a list of additional fields to add to the ElasticSearch document.  
FriendlyNameUser-friendly name of the application (for example "My Beautifull App"). If not null or empty it will be added to every log entry sent to ElasticSearch server (to "appName" property).  
SystemNameShort name of the application that will be used for naming ElasticSearch indices and aliases (for example "myapp").  
ThrowExceptions Gets or sets if exceptions will be rethrown. Set it to true if ElasticsearchTarget target is used within FallbackGroup target (https://github.com/NLog/NLog/wiki/FallbackGroup-target).  
Uri Gets or sets the ElasticSearch URI, can be multiple comma separated.
