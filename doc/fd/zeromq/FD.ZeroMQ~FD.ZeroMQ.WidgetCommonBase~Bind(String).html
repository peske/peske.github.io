Endpoint to bind to. It can be one of:
  • TCP endpoint (i.e. "tcp://1.1.1.1:1111");
  • In-process endpoint (i.e. "inproc://somename");
  • Inter process endpoint (i.e. "ipc://somename"). Note that inter-process endpoint is not supported on Windows.
